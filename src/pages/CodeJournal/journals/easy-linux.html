Linux Commands That Will Make Your Life a Lot Easier

By Erik H Robertson
Published February 19, 2023

I've always been a big fan of learning Linux Command Line. Starting out with photography, I was always curious if there we're ways I could streamline tasks. Lately I've been digging for new ways that I can tap into Linux to return the information that I want. So, here's a few Linux Commands that I've found that make life a lot easier.

Search for all PID's with Webcam Access (MacOS) 

            sudo lsof | grep "AppleCamera\|iSight" | awk '{print $2}' | sort -u

This command will use lsof to list all open files, then filter the results to only show processes that have opened a webcam device file (either AppleCamera or iSight on macOS). The awk command will extract the second field (which is the PID) from the output, and the sort -u command will sort the PID's and remove duplicates.

Iterate actions over multiple files by using "for"

This is a great method for instances when you need to perform actions over batches of files that would be otherwise cumbersome if performed individually. 

          for file in *.txt; do 
          mv -- "$file" "${file%.txt}.text"
          done
          
          # Rename all *.txt to *.text

Check all IP's on the Current Network

Joining a public network? This is a great way to see all of your compadres who are on the same network that you're connected to. Also, the address that's in position 0, is usually the IP address of the router itself. If you copy it and paste it into the browser, it will usually lead you right to the router login page, better known as the router management interface. 

            arp -a -ll

Transfer files from Local Directory to Remote Directory

Who needs WebFTP when you can just use SSH? 

          # transfer all files recursively from [local] to [remote destination]

            scp -r ./mylocalfolder/* myremoteusername@myserveraddress.com:~/destination/of-your-choice
